# Documentation Assets

## Banner Image

Place your banner image here as `banner.png` or `banner.jpg`.

### Recommended Specifications

- **Dimensions**: 1200x400px (3:1 ratio)
- **Format**: PNG or JPG
- **Size**: < 500KB for fast loading
- **Content**:
  - Project name/logo
  - Tagline or description
  - Optional: Visual representation of progressive disclosure concept

### How to Add

1. Create or design your banner image
2. Save it in this directory as `banner.png`
3. Uncomment the banner line in the main README.md:

```markdown
![Agent Skills Banner](./docs/banner.png)
```

### Design Ideas

- Use a gradient background (modern, clean)
- Include icons representing different skills (languages, frameworks)

- [Figma](https://www.figma.com) - Professional design tool
- [Photopea](https://www.photopea.com) - Free Photoshop alternative
- [Bannerbear](https://www.bannerbear.com) - API-based banner generation
